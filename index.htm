Lesson 1: Moving Around
Move with the arrow keys. Go touch the green thing!

mouse

  • Pan by dragging the canvas, while holding the mouse button down.
  • Zoom by scrolling with the mousewheel, or using the two-finger trackpad up/down gesture, or whatevs.

keyboard

  • s: Select. Hold the key down and move the pointer to create a selection box, and release the key to select all the highlighted parts and jacks. Or just tap the key to select whatever you are pointing at. The editor remembers your previous selections, which is strange but powerful.
    Nerds: It's selections.push()
  • shift + s: Unselect. The current selection is forgotten, and the previous selection becomes the current one.
    Nerds: That's selections.pop()
  • a: Add the current selection to the previous selection.
    My people: Think push(add(pop(), pop())
  • shift + a: Subtract the current selection from the previous selection.
    You know, push(subtract(pop(), pop()))
  • l: Link any input and output jacks in the current selection. Input jacks are on top of parts. Output jacks are on the bottom.
  • delete or backspace: Delete parts in the current selection, or delete links to selected jacks.
  • c: Copy the current selection to clipboard.
  • v: Paste. Hold the key down and move the pointer to position the clip, and release the key to drop the clip in place.
  • z: Undo
  • shift + z: Redo
  • m: Toggle the Clip Menu
